1. Drag and drop UEFI (Aptio V only) in the latest A version (e.g. A59) of UEFITool
2. Find Setup, right-click PE32 image section (image) and Extract as is... -> SCT file
3. Find AMITSESetupData, right-click setupdata (image) and Extract body... -> BIN file
4. Convert SCT file with Universal IFR Extractor (e.g. IRFExtractor_0.3.6_win.zip) -> TXT file
5. Drag and drop BIN and TXT files here -> JSON file
6. Possible modifications in data.json: Access Level, Failsafe, Optimal (don't touch anything else)
7. Drag and drop modified JSON and BIN files here -> modified BIN file
8. Drag and drop UEFI in the latest numeric version (e.g. 0.28.0) of UEFITool
9. Find AMITSESetupData, right-click the Freeform subtype GUID (image) and Replace body... with the modified BIN file
10. Top left: File -> Save image file...
